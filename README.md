# 🛡️ Fedora 43 Power-Up & Security Tool

Este script Bash foi desenvolvido para automatizar o setup inicial, a otimização de performance e o endurecimento (hardening) de sistemas Fedora 43. Ele é focado em desenvolvedores, estudantes de segurança e profissionais de TI que buscam um ambiente ágil e protegido.

## 🚀 Funcionalidades

- **Otimização DNF:** Configuração de downloads paralelos e seleção de mirrors mais rápidos.
- **Hardening de Segurança:** Verificação de status do SELinux e ativação automática do Firewalld.
- **Ambiente de Virtualização:** Instalação completa do KVM/QEMU e Virt-Manager (ideal para laboratórios de Whonix).
- **Auditoria de Rede:** Módulo integrado para visualização de portas e serviços em escuta.
- **Sistema de Logs:** Registro detalhado de todas as operações para auditoria e troubleshooting.

## 🎓 Contexto Acadêmico e Profissional

Este projeto integra conhecimentos práticos e teóricos adquiridos em:
* **Ciência da Computação (UFCAT):** Aplicação de conceitos de Sistemas Operacionais 1 e Algoritmos.
* **Sistemas de Informação (Uniasselvi):** Foco em governança e segurança de dados.
* **Manutenção Técnica:** Ferramenta utilizada para padronizar e agilizar o setup de máquinas de alto desempenho.

## 🛠️ Como usar

1. Clone o repositório:
   git clone https://github.com/Heartras/fedora-security-tool.git

2. Dê permissão de execução:
   chmod +x fedora_boost.sh

3. Execute como root:
   sudo ./fedora_boost.sh
